# LoansForSurgeons

Niche lead-gen flagship for loansforsurgeons.com — an AdaptLend service. Static site, no build step.

Leads insert into the shared AdaptLend Supabase `leads` table (attribution: matched_product suffix `— LoansForSurgeons`); a branded welcome email is sent via the `send-welcome-sites` Supabase edge function (Resend, from matches@adaptlend.com, CC malloo@westcapitallending.com).
